# Contributing
Roles:
  DevOps Architect:
    Configures and maintains CI/CD pipeline and performs rollbacks.
  Product Owner:
    Pushes release tags and requirements gathering.
  Cybersecurity Administrator:
    Configures and maintains security code review automated scans and reviews commits for vulnerabilities.
  Senior Developer:
    Makes, reviews commits and makes application architecture decisions.
  Junior Developer:
    Makes and reviews commits.
